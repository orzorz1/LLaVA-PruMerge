# 环境与资源配置

本文件只讲三件事：**环境、模型、数据集路径**。如何推理和训练请看 `使用与训练指南.md`。

---

## 一、基础环境

- **克隆代码**

```bash
git clone https://github.com/42Shawn/LLaVA-PruMerge.git
cd LLaVA-PruMerge
```

- **Conda 环境**

```bash
conda create -n llava-prumerge python=3.10 -y
conda activate llava-prumerge
```

- **安装依赖**

```bash
pip install -e .
```

- **安装 PyTorch（示例，根据你 CUDA 调整）**

```bash
pip install "torch==2.1.0" "torchvision==0.16.0" --index-url https://download.pytorch.org/whl/cu118
```

> 用 `nvidia-smi` 看 CUDA 版本，然后到 PyTorch 官网选对应安装命令。

## 二、模型与 LoRA 目录

### 1. 原始 LLaVA 模型（推荐直接用官方 1.5）

以 7B 为例，本地放在 `/data/models/llava-v1.5-7b`：

```bash
mkdir -p /data/models
cd /data/models

huggingface-cli download liuhaotian/llava-v1.5-7b \
  --local-dir llava-v1.5-7b \
  --local-dir-use-symlinks False
```

之后在脚本里统一使用：

```bash
--model-path /data/models/llava-v1.5-7b
```

> 如果你想自己从 Vicuna + 预训练 + 指令微调一步步训练出 LLaVA，可再看 `docs/Model_Zoo.md` 和 LLaVA 原仓库文档。

### 2. PruMerge LoRA（本项目的关键权重）

在项目根目录下创建并下载：

```bash
cd /home/pangu/lhx/LLaVA-PruMerge   # 按你实际路径改

mkdir -p checkpoints/llava-v1.5-7b-lora-prunemerge

export HF_ENDPOINT=https://hf-mirror.com  # 如需镜像，可选

huggingface-cli download \
  yuzhang/llava-prumerge-vicuna-7b-v1.5-lora \
  --local-dir checkpoints/llava-v1.5-7b-lora-prunemerge \
  --local-dir-use-symlinks False
```

下载完成后检查目录：

```bash
ls -la checkpoints/llava-v1.5-7b-lora-prunemerge/
```

应能看到至少：

- `adapter_config.json`
- `adapter_model.bin` 或 `adapter_model.safetensors`

在评测脚本中通常写成：

```bash
--model-base lmsys/vicuna-7b-v1.5
--model-path checkpoints/llava-v1.5-7b-lora-prunemerge
```

> 也可以额外下载 `yuzhang/llava-prumerge-plus-vicuna-7b-v1.5-lora` 到另一个目录，切换 LoRA 路径即可体验 PruMerge+。

## 三、评测公共结构（eval.zip）

所有评测数据都挂在 `playground/data/eval/` 下。先下载 LLaVA 官方提供的结构模版：

1. 打开本仓库 `docs/Evaluation.md` 中的 `eval.zip` 链接。  
2. 下载后在项目根目录解压：

```bash
cd /home/pangu/lhx/LLaVA-PruMerge
mkdir -p playground/data
unzip /你下载好的/eval.zip -d playground/data
```

解压后大致为：

```text
playground/data/eval/
  vqav2/
  gqa/
  textvqa/
  vizwiz/
  ...
```

下面各任务只需要再补“官方图像 + 标注文件”即可。

## 五、最小跑通 checklist（配置视角）

1. 按「一、基础环境」配置 Conda + PyTorch。  
2. 下载 `liuhaotian/llava-v1.5-7b` 到 `/data/models/llava-v1.5-7b`。  
3. 下载 `yuzhang/llava-prumerge-vicuna-7b-v1.5-lora` 到 `checkpoints/llava-v1.5-7b-lora-prunemerge`。  
4. 下载并解压 `eval.zip` 到 `playground/data/eval/`。  
5. 只准备 TextVQA：放好 `TextVQA_0.5.1_val.json`、`train_images/`，并建好 `answers/` 目录。  
6. 确认 `scripts/v1_5/eval/textvqa.sh` 中的 `--model-base` 和 `--model-path` 指向你刚才的模型/LoRA 目录。  

做到这一步，环境与资源就全部 OK，可以看 `使用与训练指南.md` 里的命令来跑推理和训练了。
